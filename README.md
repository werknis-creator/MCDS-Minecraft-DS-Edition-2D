# MCDS-Minecraft-DS-Edition-2D
Minecraft 2D clone for Nintendo DS
1. Put all files on the microSD card in the R4 adapter or SD card in 3DS or 2DS
2. Run files from R4 adapter or SD cards
3. Run "LUA code starter.nds"
4. Go to lua folder and run MCDSV01.lua
5. Enjoy the game and recommend it to others!
